# EasyOCR 1.0.0

EasyOCR is a lightweight privacy-first Android image editor for copying text from screenshots.

Highlights:

- Opens from Android screenshot/image editor intents.
- Runs OCR locally on-device.
- Supports selectable OCR text, crop, document perspective crop, rotate, drawing, save copy, and share.
- No internet permission, analytics, tracking, ads, or cloud OCR.

Install the APK from the release assets.
